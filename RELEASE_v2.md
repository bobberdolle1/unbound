# 🚀 Инструкция по выпуску релиза UNBOUND v2.0.0

## Автоматический выпуск через GitHub Actions

Самый простой способ — использовать CI/CD:

1. **Запуште тег**: `git tag -a v2.0.0 -m "v2.0.0" && git push origin v2.0.0`
2. **GitHub Actions** автоматически соберёт все платформы и создаст релиз
3. **Проверьте черновик** на https://github.com/bobberdolle1/unbound/releases

## Ручной выпуск (Windows)

```powershell
# Из корня репозитория
.\release_v2.bat
```

## Ручной выпуск (пошагово)

### 1. Сборка Windows

```powershell
wails build -clean -o unbound.exe
# Бинарник: build/bin/unbound.exe
```

### 2. Создание релиза

```powershell
gh release create v2.0.0 `
  --title "Unbound v2.0.0: Глобальное обновление" `
  --draft `
  --generate-notes `
  build/bin/unbound.exe
```

### 3. Тег и пуш

```powershell
git tag -a v2.0.0 -m "Unbound v2.0.0"
git push origin v2.0.0
git push origin main
```
