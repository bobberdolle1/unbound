# Заметки о выпуске UNBOUND v2.0.0

## МАЖОРНЫЕ ВОЗМОЖНОСТИ

- **Мультиплатформенная экосистема** — Полная поддержка Windows, macOS, Linux, Android, iOS, OpenWrt, браузеров, Smart TV, Steam Deck
- **Автоподбор V2** — Параллельный сканер с таймаутом 5 секунд на зонд
- **Системный трей** — Полноценная интеграция с треем, сворачивание вместо выхода
- **Обновлённый интерфейс** — Современный дизайн с мониторингом в реальном времени
- **Повышенная безопасность** — HideWindow на всех дочерних процессах
- **Полная русификация** — Весь UI, уведомления, документация на русском языке
