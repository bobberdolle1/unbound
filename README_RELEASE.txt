# UNBOUND v1.0.4

Программа для обхода DPI-блокировок на Windows (YouTube, Discord, Telegram и другие).
Основана на движке zapret2 (winws2) от bol-van.

## Требования
- Windows 10 / 11 (x64)
- Права администратора (обязательно!)

## Запуск
1. Нажмите правой кнопкой на **unbound.exe**
2. Выберите «Запуск от имени администратора»
3. Выберите профиль или нажмите «Автоподбор»
4. Нажмите «ПОДКЛЮЧИТЬ!»

## Первый запуск
Рекомендуется нажать «Автоподбор» — программа сама протестирует все профили
и выберет оптимальный для вашего провайдера.

## Что встроено внутри exe
Все зависимости (winws2.exe, WinDivert, Lua-скрипты, списки хостов) встроены
в unbound.exe через embed. Никаких дополнительных файлов не требуется.

## Конфликты
Если используете другой DPI-обходчик (GoodbyeDPI, старый Zapret, Cloudflare WARP),
Unbound предупредит и предложит завершить конфликтующие процессы.
