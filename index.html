macOS

Движок: SpoofDPI (SOCKS5-прокси)

Не требует root при запуске — эскалация через osascript (Touch ID/пароль) только для применения системного прокси через networksetup. Автозапуск через launchd. Graceful shutdown предотвращает потерю интернета при выходе. Для продвинутых: можно использовать nfqws напрямую с pf (Packet Filter) — divert-packet port 700 перенаправляет трафик, требует root.

Linux

Движок: nfqws + iptables/nftables NFQUEUE

Go-обёртка автоматизирует запуск nfqws с iptables mangle-правилами (connbytes 1:6, mark exclusion 0x40000000). Также доступен отдельный Rust CLI (unbound-cli) с поддержкой nftables и systemd-сервиса. Требует root. Для прямого запуска nfqws: nfqws --port=700 --dpi-desync=multidisorder + iptables -j NFQUEUE --queue-num 200. Пакеты: PKGBUILD (Arch), deb (Debian/Ubuntu), rpm (Fedora/RHEL).

Движок: tpws (SCPreferences SOCKS-прокси)

Работает даже на iPhone 4s с iOS 6.1.3 (ARMv7)! Портированный tpws с kqueue/epoll shim, системный прокси через SCPreferences, скевоморфный интерфейс в стиле iOS 6. Частые вопросы

Частые вопросы

Что такое Unbound и как он работает?

Unbound — бесплатная программа с открытым кодом для обхода DPI-блокировок. Использует движок winws2 для манипуляции TCP-пакетами — TCP десинхронизация и перестановка пакетов (Multidisorder) — чтобы системы DPI не могли анализировать и блокировать ваш трафик к YouTube, Discord, Twitter и Instagram.

Чем это отличается от VPN?

VPN маршрутизирует весь трафик через иностранный сервер, что замедляет соединение. Unbound модифицирует только пакеты к заблокированным доменам — банки, госучреждения и разблокированные сайты работают как обычно. Нулевая потеря скорости для обычного серфинга.

Зачем нужны права администратора?

Unbound использует драйвер WinDivert для перехвата и модификации сетевых пакетов на уровне ядра. Это требует прав администратора. Программа регистрируется в планировщике задач Windows для тихого автозапуска без повторных запросов UAC.

Работает ли Unbound на macOS и Linux?

Да! На macOS Unbound использует SpoofDPI (SOCKS5-прокси) с системной настройкой. На Linux используется nfqws с iptables/nftables NFQUEUE. Для десктопа Go-обёртка автоматизирует запуск, также доступен отдельный Rust CLI.

А что насчёт iOS 6 и iPhone 4s?

Да! Существует специальная legacy-версия для iOS 6.1.3+ (iPhone 4s, ARMv7). Она использует портированный движок tpws с совместимостью kqueue, системный SOCKS-прокси и скевоморфный интерфейс. Требуется джейлбрейк (Cydia Substrate).

Это эксклюзивная тема в обновлении v2.5.0, использующая экстремальное размытие фона и прозрачность. Она вдохновлена дизайном macOS Sonoma и iOS 17/18, создавая эффект "живого стекла" на вашем рабочем столе.
